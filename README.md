A command-line banking application that supports basic account operations and transaction tracking.

## Installation
```
cd banking-system
python main.py
```

## Usage
Run With main.py

### Menu Options
*When Logged Out:*
1. Login - Access existing account
2. Create Account - Create new account
3. Exit

*When Logged In:*
1. Deposit - Add funds
2. Withdraw - Remove funds 
3. Logout
